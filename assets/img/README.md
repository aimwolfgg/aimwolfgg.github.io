# Картинки сайта AimWolf

Сюда положить:

- **`logo.png`** — аватарка канала (low-poly волк, белая морда, оранжевый акцент). Используется как favicon, в шапке и футере. Размер: квадрат, идеально 256×256 или 512×512 PNG, прозрачный фон НЕ нужен (волк уже на тёмном фоне).
- **`og-banner.png`** — баннер для превью сайта при шеринге в TG/VK. Размер: **1200×630** (стандарт Open Graph). Бери тот же баннер, что в TG-канале (BRAWL STARS · DISCORD · FORTNITE / NO BLOCKS), но обрежь до 1200×630 если нужно.

## Что сделать перед заливкой

1. Прогнать каждую картинку через **squoosh.app** (mozjpeg или WebP, качество 85). Файлы должны быть < 200 КБ каждый.
2. Имена файлов — **строго в нижнем регистре**, без пробелов и кириллицы.

После добавления картинок сайт сразу можно тестировать локально:
`cd sites/aimwolf && python -m http.server 8000` → открыть `http://localhost:8000`.
